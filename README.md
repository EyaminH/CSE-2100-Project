# Super_Mario_Bros_Project

A Simple Clone of Super Mario Bros, World 1-1 through 1-2. Made with Unity.

***
### Controls
* Move: Left/Right
* Crouch: Down
* Jump: Z
* Dash: Hold X while moving or jumping
* Pause/Unpause game: Enter/return
